# ร้านชากาแฟ — POS

ระบบขายหน้าร้าน (Point of Sale) สำหรับร้านชากาแฟ ภาษาไทย ใช้เงินบาท (฿)

## วิธีเปิดใช้

1. **เปิดในเบราว์เซอร์** — ดับเบิลคลิก `index.html` หรือลากเข้าเบราว์เซอร์
2. **ใช้เซิร์ฟเวอร์ท้องถิ่น (แนะนำ)** — รันคำสั่งในโฟลเดอร์นี้:
   ```bash
   npx serve .
   ```
   แล้วเปิด URL ที่แสดง (เช่น http://localhost:3000)

## ฟีเจอร์

- **หมวด:** กาแฟ, ชา
- **ตะกร้า:** เพิ่มรายการ เปลี่ยนจำนวน (+/−) ล้างรายการ
- **ยอด:** รวมทั้งหมด (บาท) ไม่มีภาษี
- **สั่งซื้อ:** กดสั่งซื้อ → ใบเสร็จ
- **พิมพ์:** ดูและพิมพ์ใบเสร็จ
- **ออเดอร์ใหม่:** เพิ่มหมายเลขออเดอร์และล้างตะกร้า
- **ถือรายการ:** ถือออเดอร์ปัจจุบันและเริ่มออเดอร์ใหม่

## แอดมิน

- เปิด **admin.html** (หรือคลิกลิงก์ "แอดมิน" ที่หัว POS)
- **เข้าสู่ระบบ:** ชื่อผู้ใช้ `Mixzaza` / รหัสผ่าน `121212`
- **สถานะออเดอร์:** ออเดอร์ใหม่จะแสดง "รอจ่าย" — กดปุ่ม **จ่ายแล้ว** เมื่อลูกค้าชำระเงินแล้ว
- **สรุปยอดวันนี้:** แสดงจำนวนออเดอร์ที่จ่ายแล้วและยอดรวมของวัน (อัปเดตเมื่อกดจ่ายแล้ว)
- **รายการล่าสุด:** รายการสั่งซื้อทั้งหมด (ล่าสุดอยู่ด้านบน)
- **ย้อนหลัง 1 เดือน:** สรุปยอดแยกตามวัน (เฉพาะออเดอร์ที่จ่ายแล้ว) แสดงยอดรวมต่อวันและรายการออเดอร์
- **ล้างสินค้า:** ปุ่มล้างรายการสั่งซื้อทั้งหมดและรีเซ็ตหมายเลขออเดอร์เป็น 1001 — ต้องใส่รหัส (ใช้รหัสเดียวกับ Login คือ `121212`) แล้วกดยืนยัน
- **ออกจากระบบ** จะออกจากเซสชัน

## แก้ไข/ปรับแต่ง

- **สินค้า:** แก้ไข object `products` ใน `app.js` เพื่อเพิ่ม/ลบเมนูหรือเปลี่ยนราคา (หน่วยเป็นบาท)
